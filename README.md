# Base Prediction Market

Simple onchain ETH price prediction market built on Base.

## Features

- Daily ETH price predictions
- Onchain submissions
- Small entry fee
- Public prediction history
- Base compatible

## Stack

- Solidity
- Hardhat
- OpenZeppelin
- Base
